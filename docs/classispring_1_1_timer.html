시간 측정을 죵료합니다.

매개변수
param시간 측정 구분자(이 값은 char,int,float,double,string)을 지원합니다.
반환값
TimerElement구조체를 반환합니다. 이 구조체는 curr,avg,accu 를 가지고 있으며 현재 구간 시간, 평균 구간 시간, 누적 구간 시간 입니다.
경고
이 함수는 반드시 Tick이 불려진다음 호출되야 합니다.
Remarks
ispring::Timer::Tick("for");
for (int i = 0; i < 5; i++) {
ispring::Timer::Tick(1234);
Sleep(100);
std::cout << "elem : " << ispring::Timer::Tock(1234).accu << std::endl;
}
std::cout << "for : " << ispring::Timer::Tock("for").curr << std::endl;
